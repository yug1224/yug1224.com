`apt-get install git-core` で入る Git が1.7と古かったので、最新の Git 1.8.5.3 を入れるためのメモ。

	$ sudo apt-add-repository ppa:git-core/ppa
	$ sudo apt-get update
	$ sudo apt-get install git-core -y

	$ git --version
	git version 1.8.5.3

ppa のリポジトリを登録して、install するだけ。簡単。
